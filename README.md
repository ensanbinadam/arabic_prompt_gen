# مولِّد البرومبتات العربيَّة والإسلاميَّة للتَّهاني الفاخرة

أداة ويب عربية تساعد على إنشاء برومبتات جاهزة لمولدات الصور مثل **Nano Banana 2**، بهدف إنتاج تصاميم تهنئة عربية فاخرة بمخطوطات فنية أنيقة، مع دعم:

- العبارتين الرئيسيتين
- سطرَي التذييل
- اسم الشخص
- اختيار الدولة
- خلفية علم مائي مرفرف
- عنصر سفلي تلقائي مناسب للدولة

## المميزات

- يدعم اللغة العربية بشكل منظم وواضح
- يولّد برومبتًا عربيًا فنيًا جاهزًا للنسخ
- يحافظ على طابع فاخر وهادئ ومتوازن
- يجعل **المملكة العربية السعودية** تستخدم **سيفًا ذهبيًا أفقيًا** تلقائيًا
- يجعل **بقية الدول** تستخدم **فاصلًا ذهبيًا أفقيًا** تلقائيًا
- يدعم خلفية **علم مائي مرفرف** للدولة المختارة
- لا يحتاج إلى API
- سهل الاستخدام والنشر

## طريقة الاستخدام

1. افتح ملف الأداة في المتصفح.
2. اكتب:
   - العبارة الرئيسية الأولى
   - العبارة الرئيسية الثانية
   - سطر التذييل الأول
   - سطر التذييل الثاني
   - اسم الشخص
3. اختر الدولة من القائمة.
4. اضغط زر **توليد البرومبت**.
5. انسخ البرومبت الناتج.
6. الصقه في **Nano Banana 2** أو أي مولد صور يدعم البرومبتات العربية.

## منطق العنصر السفلي

- عند اختيار **المملكة العربية السعودية**:
  - يظهر **سيف ذهبي أفقي** تلقائيًا

- عند اختيار **أي دولة أخرى**:
  - يظهر **فاصل ذهبي أفقي رفيع** تلقائيًا

## ملاحظات مهمة

- الأداة مخصصة لتوليد **البرومبت النصي** باحتراف.
- جودة النتيجة النهائية تعتمد أيضًا على استجابة مولد الصور نفسه.
- تم تصميم المنطق الداخلي للأداة للحفاظ على الاتساق الجمالي والرمزي بين الدول.
